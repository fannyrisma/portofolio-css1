Design Portal Lomba

Lihat Project

Project ini merupakan perancangan UI/UX Design untuk sebuah portal lomba berbasis web, yang dibuat menggunakan Figma sebagai tools utama. Portal ini dirancang untuk memudahkan mahasiswa dalam mencari, mendaftar, dan memantau berbagai lomba akademik maupun non-akademik secara online. Project ini dibuat untuk memenuhi tugas akhir mata kuliah Design Pengalaman Pengguna (DPP) sekaligus sebagai bagian dari portofolio pribadi di bidang UI/UX Design.
